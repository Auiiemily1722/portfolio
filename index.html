主推项目

Clippin

Windows 剪贴板管理工具

桌面应用

自动记录剪贴板内容，支持类型识别、历史搜索、置顶管理、导出文档和桌面端打包发布。

  • Electron
  • HTML/CSS/JS
  • sql.js
  • clipboard-event
  • pdfkit
  • electron-builder
GitHub 开源仓库

解决的问题：频繁复制文本、链接、代码片段时，系统剪贴板只能保留最近一次内容，难以追踪、搜索和复用。

项目亮点：打通了从产品构思、功能设计、桌面端开发、Windows 打包发布到 GitHub 开源的完整流程，项目具备可安装、可使用、可审阅的交付形态。

SignalOS

AI 信息工作流 / 智能信息筛选系统

信息工作流

整合 RSS 与播客信息源，通过分阶段筛选、模型分析、任务调度和控制台配置生成每日情报简报。

  • Python
  • FastAPI
  • APScheduler
  • feedparser
  • Next.js
  • React
  • TypeScript
  • Tailwind

解决的问题：自动查询科技圈每日动态，把分散新闻整理成一份简报，让我每天花约五分钟就能了解科技圈的重要变化。

项目亮点：包含数据源开关、Stage1 初筛、Stage2 深度分析、模型路由、定时刷新、邮件预览和日报生成等完整工作流。

ProjectTest

自动化采集与数据处理平台

自动化平台

支持多地区任务配置、自动采集、进度追踪、日志监控，并输出 CSV / Excel 与 SQLite .db 结果。

  • Flask
  • Selenium
  • undetected-chromedriver
  • pandas
  • requests
  • SQLite
  • HTML/CSS/JS

解决的问题：自动化、大规模获取潜在客户联系方式，减少人工搜索、复制、整理和复查的重复劳动。

项目亮点：封装地区配置、采集任务、进度状态、日志输出、SQLite 数据库写入，以及 CSV / Excel 表格结果导出。
