# 我的作品集

这是一个可以直接部署到 GitHub Pages、Cloudflare Pages 或 Vercel 的静态作品集页面。

入口文件：

```text
index.html
```

## 展示项目

- Clippin：Windows 剪贴板管理工具，已开源。
- SignalOS：AI 信息工作流 / 智能信息筛选系统，使用截图作为项目凭证。
- ProjectTest：自动化采集与数据处理平台，使用截图和输出结果作为项目凭证。

## 素材目录

所有页面素材都使用相对路径：

```text
assets/projects/clippin/cover.png
assets/projects/clippin/demo.gif
assets/projects/signalos/dashboard.jpg
assets/projects/signalos/report.jpg
assets/projects/projecttest/dashboard.png
assets/projects/projecttest/output.png
```

## 部署说明

这是纯 HTML / CSS / JavaScript 项目，不依赖后端服务，也不需要构建步骤。

部署时上传以下内容即可：

```text
index.html
styles.css
script.js
assets/
```
